# youtube_project_rookies
Sk Shieldus Rookies 미니 프로젝트

## 주제 
* 특정 음악장르 유튜버를 여려명 지정하고, 영상에서 볼 수 있는 속성들를 사용해, 결산 레포트 만들기
* 최종적으로 본인 유튜브(까까들)와 상관관계 분석

## 속성값

### 국가
- 국내

### 음악 장르
- 발라드

### YouTube 자료
- 제목, 구독자, 조회수, 좋아요, 싫어요 등등
